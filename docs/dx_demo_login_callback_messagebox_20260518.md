# DX Demo Login Callback MessageBox

Date: 2026-05-18

## Goal

Make the SDK login callback visible in `DXLoginDemo`.

Previously the DX demo registered `ShowLoginDialog(&loginCallback, 9001, 0)`, but callback details were mainly written to debugger output and the window title. That made it easy to miss whether the callback was actually fired.

## Change

`DXLoginDemo` now shows a `MessageBoxW` when the login callback returns.

The message includes:

- `errorCode`
- `userid`, using `SDOLLoginResult::Sndaid`
- `sessionid`, using `SDOLLoginResult::SessionId`

The same message text is generated by `dxDemoLoginCallbackMessage` so it is covered by `sdologin_tests`.
